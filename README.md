## 📁 Como baixar o projeto

```bash

# Clonar o repositório
$ git clone https://github.com/Leandro-Pinho/server

# Entrar no diretório
$ cd server

# Instalar as dependências
$ npm install

# Iniciar o projeto
$ npm run dev


```
